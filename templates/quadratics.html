Rozwiązywanie równań kwadratowych

Równanie kwadratowe to równanie postaci \(ax^2 + bx + c = 0,\) gdzie \(a, b, c \in \mathbb{R}\) i \(a \ne 0\). Przez rozwiązanie takiego równania rozumie się wyznaczenie wszystkich \(x \in \mathbb{R}\) spełniających to równanie. Przykładowe równanie kwadratowe \(x^2 + 3x - 4 = 0\).

W zależności od współczynników \(a, b, c \) równanie może mieć zero, jedno lub dwa rozwiązania. Do wyznaczenia liczby rozwiązań przydaje się narzędzie nazwane wyróżnikiem trójmianu kwadratowego oznaczane \(\Delta\), zdefiniowane jako \(\Delta = b^2 - 4ac\). Liczba rozwiązań równania zależy od znaku wyróżnika:

  1. Jeżeli \(\Delta \gt 0\), to równanie ma dwa rozwiązania rzeczywiste:
    $$x_1 = \frac{-b - \sqrt{\Delta}}{2a}, \quad x_2 = \frac{-b + \sqrt{\Delta}}{2a}.$$
  2. Jeżeli \(\Delta = 0\), to równanie ma jedno rozwiązanie rzeczywiste
    $$x = \frac{-b}{2a}.$$
  3. Jeżeli \(\Delta \lt 0\), to równanie nie ma rozwiązań rzeczywistych.

Przykład 1. Rozwiąż równanie \(x^2 + 3x - 4 = 0\).
Rozwiązywanie równania zaczynamy od policzenia wyróżnika

$$\Delta = b^2 - 4ac = 3^2 -4(1)(-4) = 9 + 16 = 25.$$

Delta wyszła dodatnia, zatem równanie ma dwa rzeczywiste rozwiązania. Możemy je wyliczyć używając odpowiednich wzorów

$$x_1 = \frac{-b - \sqrt{\Delta}}{2a} = \frac{-3 - \sqrt{25}}{2} = \frac{-3 - 5}{2} = -4,$$
$$x_2 = \frac{-b + \sqrt{\Delta}}{2a} = \frac{-3 + \sqrt{25}}{2} = \frac{-3 + 5}{2} = 1.$$

Stąd dostajemy odpowiedź \(x = -4 \lor x = 1\).

Przykład 2. Rozwiąż równanie \(2x^2 + 6x + 5 = 0\).
Liczymy deltę

$$\Delta = b^2 - 4ac = 6^2 - 4(2)(5) = 36 - 40 = -4.$$

Wyróżnik okazał się ujemny, zatem równanie nie ma rozwiązań w zbiorze liczb rzeczywistych.

Rozwiązywanie nierówności kwadratowych

Do rozwiązania nierówności kwadratowych potrzebne są umiejętności rozwiązywania równań kwadratowych i rysowania wykresów funkcji. Nierówność kwadratowa może przyjmować jedną z czterech postaci:

  1. \(ax^2 + bx + c \lt 0,\)
  2. \(ax^2 + bx + c \le 0,\)
  3. \(ax^2 + bx + c \ge 0,\)
  4. \(ax^2 + bx + c \gt 0.\)

Rozwiązaniem nierówności kwadratowej jest zbiór takich \(x \in \mathbb{R},\) które spełniają podaną nierówność.

Metoda rozwiązywania nierówności kwadratowych:

  1. Rozwiązać równanie \(ax^2 + bx + c = 0\),
  2. Naszkicować przybliżony wykres funkcji \(f(x) = ax^2 + bx + c\) z zaznaczonymi miejscami zerowymi,
  3. Z wykresu odczytać wynik.

Przykład 1. Rozwiąż nierówność \(3x^2 + 2x - 5 \lt 0.\)
W pierwszym kroku znajdujemy rozwiązania równania \(3x^2 + 2x - 5 = 0.\) W tym celu liczymy wyróżnik

$$\Delta = b^2 - 4ac = 2^2 - 4(3)(-5) = 4 + 60 = 64.$$

Delta okazała się dodatnia, czyli równanie ma dwa rozwiązania:

$$x_1 = \frac{-b - \sqrt{\Delta}}{2a} = \frac{-2 - \sqrt{64}}{6} = \frac{-2 - 8}{6} = -\frac{5}{3},$$
$$x_2 = \frac{-b + \sqrt{\Delta}}{2a} = \frac{-2 + \sqrt{64}}{6} = \frac{-2 + 8}{6} = 1.$$

Następnie szkicujemy wzór funkcji \(f(x) = 3x^2 + 2x - 5\) i znajdujemy zbiór takich \(x\), dla których \(f(x) \lt 0.\)

Stąd dostajemy odpowiedź \(x \in [-\frac{5}{3}, 1].\)

Przykład 2. Rozwiąż nierówność \(3x^2 + 4x + 2 \ge 0.\)
Licząc deltę otrzymujemy

$$\Delta = b^2 - 4ac = 4^2 - 4(3)(2) = 16 - 24 = -8$$

Ujemny wyróżnik oznacza brak rozwiązań równania \(3x^2 + 4x + 2 = 0,\) czyli wykres funkcji \(f(x) = 3x^2 + 4x + 2\) leży w całości nad lub pod osią OX. Aby ustalić która z tych opcji zachodzi możemy zwrócić uwagę na znak współczynnika stojącego przy \(x^2\) - jeżeli jest ujemny, to wykres leży pod osią OX, jeżeli dodatni to nad osią OX. W tym przypadku współczynnik stojący przy \(x^2\) to 3, więc dla każdego \(x \in \mathbb{R} \quad f(x) \gt 0.\) Aby upewnić się, że nie popełniliśmy błędu, możemy naszkicować wykres funkcji \(f(x) = 3x^2 + 4x + 2.\)

Widzimy że w istocie dla każdego \(x \in \mathbb{R} \quad 3x^2 + 4x + 2 \gt 0,\) więc rozwiązaniem nierówności \(3x^2 + 4x + 2 \ge 0\) jest cały zbiór \(\mathbb{R}.\)

Wzory Viete'a

Jeżeli \(x_1, x_2\) są rozwiązaniami równania kwadratowego \(ax^2 + bx + c = 0,\) to zachodzą następujące wzory:

  • $$x_1 + x_2 = \frac{-b}{a};$$
  • $$x_1 \cdot x_2 = \frac{c}{a}.$$

Dowód:

$$x_1 + x_2 = \frac{-b - \sqrt{\Delta}}{2a} + \frac{-b + \sqrt{\Delta}}{2a} = \frac{-b - \sqrt{\Delta} - b + \sqrt{\Delta}}{2a} = \frac{-2b}{2a} = \frac{-b}{a};$$
$$\begin{align} x_1 \cdot x_2 &= \frac{-b - \sqrt{\Delta}}{2a} \cdot \frac{-b + \sqrt{\Delta}}{2a} = \frac{(-b - \sqrt{\Delta)}(-b + \sqrt{\Delta)}}{4a^2}\\ &= \frac{b^2 - \Delta}{4a^2} = \frac{b^2 - (b^2 - 4ac)}{4a^2} = \frac{4ac}{4a^2} = \frac{c}{a}. \end{align}$$

Wzory Viete'a są bardzo przydatne na przykład przy rozwiązywaniu równań kwadratowych z parametrem.

Przykład 1. Dla jakich wartości parametru \(m \in \mathbb{R}\) równanie \(x^2 + mx + m = 0\) ma dwa różne rozwiązania rzeczywiste przeciwnych znaków?
Podane równanie będzie miało dwa rozwiązania wtedy, gdy wyróżnik będzie większy od 0. zaczynamy więc od rozwiązania nierówności \(\Delta \gt 0.\)

$$\Delta = b^2 - 4ac = m^2 - 4m = m(m - 4).$$ $$\quad \Delta \gt 0 \Leftrightarrow m(m-4) \gt 0 \Leftrightarrow m \in (-\infty, 0) \cup (4, \infty).$$

Teraz pozostaje sprawdzić dla jakich \(m \in (-\infty, 0) \cup (4, \infty)\) \(x_1\) i \(x_2\) mają przeciwne znaki. Dwie liczby mają przeciwne znaki wtedy i tylko wtedy, gdy ich iloczyn jest ujemny. Skorzystamy w tym miejscu z poznanych wzorów Viete'a.

$$x_1 \cdot x_2 = \frac{c}{a} = \frac{m}{1} = m. \quad x_1 \cdot x_2 \lt 0 \Leftrightarrow m \lt 0.$$

Stąd otrzymujemy odpowiedź \(m \in (-\infty, 0).\)
