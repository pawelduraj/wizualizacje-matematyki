Logarytmy

Definicja logarytmu - logarytm przy podstawie \(a\) z liczby \(b\) to taka liczba \(c,\) że \(a^c = b.\) Zapis matematyczny: \(\log_{a} b = c \Leftrightarrow a^c = b.\)

Założenia:

  1. \(a \gt 0\)
  2. \(a \ne 1\)
  3. \(b \gt 0\)

Przykład 1. \(\log_{2} 16 = 4\), ponieważ \(2^4 = 16\)

Działania i operacje na logarytmach


Jeżeli \(a \gt 0,\) \(a \ne 1,\) \(b \gt 0,\) \(c \gt 0,\) to zachodzą następujące wzory:

  1. \(\log_{a} b + \log_{a} c = \log_{a} (b \cdot c);\)
  2. \(\log_{a} b - \log_{a} c = \log_{a} (\frac{b}{c});\)
  3. \(\log_{a} b^n = n \cdot \log_{a} b;\)
  4. \(a^{\log_{a} b} = b;\)
  5. \(\log_{a} b = \frac{\log_{c} b}{\log_{c} a}.\)

Dowód

  1. Niech \(\log_{a} b = x, \log_{a} c = y.\) Wtedy z definicji logarytmu \(a^x = b, a^y = c.\)
    $$b \cdot c = a^x \cdot a^y = a^{x + y} \Rightarrow \log_{a} (b \cdot c) = \log_{a} (a^{x + y}) = x + y = \log_{a} b + \log_{a} c.$$
    Czyli ostatecznie \(\log_{a} b + \log_{a} c = \log_{a} (b \cdot c).\)
  2. Niech \(\log_{a} b = x, \log_{a} c = y.\) Wtedy z definicji logarytmu \(a^x = b, a^y = c.\)
    $$\frac{b}{c} = \frac{a^x}{a^y} = a^{x - y} \Rightarrow \log_{a} (\frac{b}{c}) = \log_{a} (a^{x - y}) = x - y = \log_{a} b - \log_{a} c.$$
    Czyli ostatecznie \(\log_{a} b - \log_{a} c = \log_{a} (\frac{b}{c}).\)
  3. Niech \(\log_{a} b = x.\) Wtedy z definicji logarytmu \(a^x = b.\)
    $$a^x = b \Rightarrow a^{nx} = b^n \Rightarrow log_{a} (a^{nx}) = log_{a} (b^n) \Rightarrow nx = log_{a} (b^n) \Rightarrow n \log_{a} b = log_{a} (b^n).$$
    Czyli ostatecznie \(log_{a} (b^n) = n \log_{a} b.\)
  4. Niech \(\log_{a} b = x.\) Wtedy z definicji logarytmu \(a^x = b.\)
    $$a^{log_a b} = a^x = b.$$
    Czyli ostatecznie \(a^{\log_{a} b} = b.\)
  5. Niech \(\log_{a} b = x.\) Wtedy z definicji logarytmu \(a^x = b.\)
    $$\log_{c} b = \log_{c} (a^x) = x \log_{c} a.$$
    $$\frac{\log_{c} b}{\log_{c} a} = \frac{x \log_{c} a}{\log_{c} a} = x = \log_{a} b.$$
    Czyli ostatecznie \(\log_{a} b = \frac{\log_{c} b}{\log_{c} a}.\)

Przykład 2. Oblicz \(\frac{\log_{5} 4 + \log_{25} 4}{\log_{5} 3}.\)

$$\frac{\log_{5} 4 + \log_{25} 4}{\log_{5} 3} = \frac{\log_{5} 4 + \frac{\log_{5} 4}{\log_{5} 25}}{\log_{5} 3} = \frac{\log_{5} 4 + \frac{\log_{5} 4}{2}}{\log_{5} 3} = \frac{\frac{3}{2} \log_{5} 4}{\log_{5} 3} = \frac{\log_{5} 4^{\frac{3}{2}}}{\log_{5} 3} = \frac{\log_{5} 8}{\log_{5} 3} = \log_{3} 8$$

Przykład 3. Oblicz \(\frac{1}{2} \log_{3} 15 - \log_3 \sqrt{5}.\)

$$\frac{1}{2} \log_{3} 15 - \log_3 \sqrt{5} = \log_{3} \sqrt{15} - \log_3 \sqrt{5} = \log_{3} \sqrt{\frac{15}{5}} = \log_{3} \sqrt{3} = \frac{1}{2}$$

Przykład 4.Rozwiąż równanie \(log_{\sqrt{2}} (\log_{7} \frac{2^x + 104}{2^{x+1}}) = 0.\)
Z definicji logarytmu

$$\log_{7} \frac{2^x + 104}{2^{x+1}} = \sqrt{2}^0 = 1$$ $$\frac{2^x + 104}{2^{x+1}} = 7^1 = 7$$ $$2^x + 104 = 7(2 \cdot 2^x)$$ $$104 = 13 \cdot 2^x$$ $$2^x = 8$$ $$x = 2.$$

Odpowiedź: \(x = 2.\)
